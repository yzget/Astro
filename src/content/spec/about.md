# 关于我
嗨，我是 [蝴蝶飞]！

我不是一个喜欢长篇大论的创作者，而是一个重度网络冲浪选手和实用工具收集者。

在网上看到好玩的开源项目、通俗易懂的技术教程，或者折腾网络代理、博客搭建时的避坑指南，我都会习惯性地把它们梳理、转存到这里。

⚙️ 这里主要关注和折腾：

实用工具： GitHub 上那些能提升效率、解决痛点的免费开源软件。

网络折腾： 各种网络代理软件的配置、优化与使用心得。

建站与折腾： 博客搭建、Web 基础设施以及各种好玩的数字手艺。

📌 为什么建这个博客？
好记性不如烂笔头。把这些好东西集中在这里，第一是为了我自己随时备查，第二是如果这些被筛选过的“干货”顺便能帮到在网络另一端折腾同样技术的你，那就再好不过了。

感谢你的路过，祝你折腾愉快！
